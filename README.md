# STM32 学习笔记

这是我学习 STM32F4 的代码记录，基于 Keil MDK + HAL 库。

## 开发环境

- **主控芯片**：STM32F407ZGT6
- **IDE**：Keil MDK5
- **库**：HAL 库 + 标准外设库
- **调试器**：ST-Link / J-Link

*注：如果用的芯片或库不同，请以你实际为准，改上面这行就行。*

## 如何运行

1. 用 Keil MDK5 打开对应文件夹下的 `project.uvprojx`
2. 编译（F7），下载（F8）
3. 连接对应硬件即可看到效果

## 学习路径建议

```
3-1 LED闪烁 → 3-2 流水灯 → 3-3 蜂鸣器 → 3-4 按键
    ↓
4-1 OLED → 5-1 红外 → 5-2 编码器
    ↓
6-1 定时中断 → 6-3 PWM → 6-4 舵机 → 6-5 电机
```

按数字顺序走就行，前面是基础，后面会用到前面的知识。
